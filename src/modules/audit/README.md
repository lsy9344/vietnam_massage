# Audit Module

Owns immutable change history.

## Includes

- actor
- action
- target type
- target ID
- before state
- after state
- timestamp

## Upstream

- events from `masters`, `calls`, `settlements`, and `closing`

## Downstream

- administrator review
- close review
- traceability reports

## Does Not Own

- business decisions
- payout calculation
- source record mutation
